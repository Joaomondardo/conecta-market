# Projeto Next.js

Este é um projeto desenvolvido com Next.js

# Como executar o projeto

Primeiro, instale as dependências:

```bash
npm install
```

Depois, execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra o navegador e acesse:

```txt
http://localhost:3000
```

A aplicação será atualizada automaticamente conforme você editar os arquivos do projeto.

# Estrutura principal

Você pode começar a editar a página principal modificando o arquivo:

```txt
app/page.tsx
```
